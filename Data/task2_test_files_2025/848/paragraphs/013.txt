Each package of NOVO GLUCOSE test strip vials (2 or 4 vials to a package) has a large printed insert providing directions on how to use the NOVO GLUCOSE test strips. The insert's headline contains the words NOVO GLUCOSE STRIPS (BLOOD GLUCOSE TEST STRIPS) FOR USE WITH ALL ONE TOUCH® METRES IMPORTANT: READ THIS INSERT BEFORE USING. Immediately below is a display of the three ONE TOUCH Metres and below that display are the following words:
"The NOVO GLUCOSE Strips (blood glucose test strips) are to be used to check for changes in blood sugar levels. The test strips can be used with ONE TOUCH® Basic, ONE TOUCH® II, and ONE TOUCH® PROFILE Metres ..."
